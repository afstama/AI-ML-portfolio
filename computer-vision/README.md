# Computer Vision

**MSc Course** · KU Leuven, Faculty of Engineering Science · 2023–2024

---

## Overview

Lab assignments from the Computer Vision course at KU Leuven, covering classical image processing through deep learning-based approaches using OpenCV and PyTorch.

---

## Assignment 1 — Image & Video Processing with OpenCV

[`assignment1/`](assignment1/)

**Topics:** Real-time image processing, video frame analysis, object detection fundamentals.

- Frame extraction and manipulation from video streams
- Color space processing and filtering
- Object detection pipeline with OpenCV

**Key files:** `colorsCam.py`, `functionsImg.py`, `opencv_process_video.py`

**Skills:** OpenCV, video processing, image filtering, Python

---

## Assignment 2 — Classification with Deep Learning

[`assignment2/`](assignment2/)

**Topics:** Multi-class image classification, cross-validation, deep learning model training.

- Built and trained classification models on a custom image dataset (group project)
- Implemented k-fold cross-validation for robust evaluation
- Kaggle-style submission pipeline

**Key files:** `group15-notebook.ipynb`, `group15-notebook-k-cross-edits.ipynb`

**Skills:** PyTorch, image classification, k-fold CV, data pipelines

---

## Assignment 3 — Advanced Architectures

[`assignment3/`](assignment3/)

**Topics:** Transfer learning, exploratory data analysis, ResNet fine-tuning.

- EDA on image dataset before modeling
- ResNet-based transfer learning pipeline
- Model comparison and performance analysis

**Key files:** `exploration.ipynb`, `resnet.ipynb`, `group15_notebook.ipynb`

**Skills:** Transfer learning, ResNet, PyTorch, EDA

---

## Tech Stack

`Python` · `OpenCV` · `PyTorch` · `NumPy` · `Matplotlib`
