# Introduction to Artificial Intelligence — Labs

**BSc Course (uuui)** · University of Zagreb, FER · 2022–2023

---

## Overview

Four lab assignments covering core AI algorithms implemented from scratch in Python — search, logic, machine learning, and neural networks.

---

## Lab 1 — State Space Search & Markov Decision Processes

[`lab1/`](lab1/)

**Topics:** State space representation, informed/uninformed search, Markov Decision Processes.

- Implemented graph search over state transition models
- Modeled probabilistic transitions (e.g., course outcome simulation with failure/pass probabilities)
- MDP value iteration and policy derivation

**Skills:** Search algorithms, MDP, graph traversal, Python

---

## Lab 2 — Resolution-Based Theorem Proving

[`lab2/`](lab2/)

**Topics:** Propositional logic, resolution refutation, clause normal form.

- Implemented resolution algorithm for automated theorem proving
- Parsed and processed clause-form knowledge bases from text input
- Negation and clause simplification logic

**Key files:** `lab2.py`, `zadnji.py`

**Skills:** Logic, resolution, propositional calculus, Python

---

## Lab 3 — Decision Trees & Naive Bayes

[`lab3/`](lab3/)

**Topics:** Supervised learning, ID3 decision tree, Naive Bayes classifier.

- Implemented ID3 decision tree from scratch (information gain splitting)
- Implemented Naive Bayes classifier with Laplace smoothing
- Evaluated on CSV datasets with configurable train/test split

**Key file:** `lab3.py`

**Skills:** Decision trees, Naive Bayes, entropy, information gain, Python

---

## Lab 4 — Neural Network from Scratch

[`lab4/`](lab4/)

**Topics:** Feedforward neural networks, backpropagation, gradient descent.

- Implemented a fully connected neural network in pure NumPy
- Forward and backward pass with configurable layers and activations
- Trained and evaluated on classification tasks

**Key file:** `solution.py`

**Skills:** NumPy, backpropagation, neural network implementation from scratch
