## Projects

### 🧬 [Molecular Property Prediction via Cross-Modal Representation Fusion](molecular-property-prediction/)
**Master's Thesis · FER Zagreb · 2025 · 🏆 GDi Excellence Award**

Predicts lipophilicity (a key drug ADMET property) from molecular structure by fusing three complementary representations: Mordred descriptors (2D/3D), molecular fingerprints (MACCS, ECFP, ErG), and NLP-inspired embeddings (Mol2Vec, ChemBERTa). Uses scaffold-aware cross-validation for realistic generalization estimates.

`Python` `PyTorch` `DeepChem` `RDKit` `HuggingFace Transformers` `Scikit-Learn`
